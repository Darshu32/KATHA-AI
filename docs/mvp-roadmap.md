# MVP Roadmap

## Phase 1 foundation

- Monorepo layout for frontend, backend, and shared models
- Canonical design graph types
- Landing page and project studio shell
- FastAPI health and project bootstrap endpoints

## Phase 2 interaction

- Prompt to structured design intent
- Project creation and version snapshots
- Selection inspector and theme switching
- Estimate assumptions panel

## Phase 3 intelligence

- OpenAI-powered prompt parsing
- Retrieval-backed architecture assistant
- Local region editing
- 2D and 3D asset synchronization
